Branching Model:

1. Main : Builds come from here

2. Develop : Integration of features here

---

3. Feature: Created from "develop". 
- naming: feature/<short-description>

4. Bugfix: 
- naming: bugfix/<issue-number-or-description>

5. Hotfix: Only for urgent fixes on main.
- naming: hotfix/<short-description>

5. Art:
- naming: art/<short-description>
